# Bienvenidos a la creación de mi Portafolio
[![My Skills](https://skillicons.dev/icons?i=js,html,css)](https://skillicons.dev)

> Este es hasta el momento un gran paso para mí, el hecho de desarrollar mi propio portafolio aplicando los conocimientos que he venido recolectando es una gran satisfacción y espero así empezar a contribuir en otros proyectos.


Este proyecto :computer: está desarrollado en HTML, CSS y algo de JavaScript, con un diseño responsivo. Entre algunas cosas tiene: 

* Un menú deplegable en la cabecera cuando está en vista para pantallas móviles :iphone:.
* Otro menú desplegable en la parte inferior para ver algunos estudios :page_with_curl:.
* Un formulario para el contacto :mailbox_with_no_mail:.
* En vista mobile se configura la cabecera y otros elementos :cool:.
